# 可调横轴锁

## 中文
横轴锁栓在弹簧作用下顶住刀根。其特色是可调节弹簧预紧力，让开合阻力与回弹速度适应不同使用者。

## English
A spring-driven crossbar engages the blade tang. Adjustable spring preload lets the user tune deployment resistance and return speed.
